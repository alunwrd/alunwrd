> *A custom x86-64 OS, written in Rust*

- Custom **MBR bootloader** with hybrid UEFI/Legacy support via GRUB chainload
- Custom **libc** layer (`libmiku`) - not a musl port, written in Rust
- **PMM** (bitmap allocator), **4-level paging VMM**, **CFS scheduler**
- **VFS** with ext2/ext4/tmpfs/devfs/procfs
- **TLS 1.2/1.3** networking stack
- Custom **dynamic linker** (`ld-miku`)
- ATA driver, framebuffer console, PCI scanner

```
boot → kernel → drivers → vfs → scheduler → libc → userspace
  └── all in Rust. no compromise.
```
